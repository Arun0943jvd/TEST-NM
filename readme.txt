Steps for Running the code

1)After downloading the Folder or Clone.
2)Download and Install node by clicking on .exe file.( https://nodejs.org/en/)
3)open cmd and go to the root folder of the test and then type 
  npm install
4)When the 3rd step is done.Then you will be able to see the node_modules folder created by it.
5)Once the 4th step is done then you can run the webpack in the cmd window .
then it will generate a build folder once it has generated it .
6)run the index.html .
7)open dev tool of the browser and click on search or display button to see the output of the ajax call. (As i have taken by own ajax i have the ajax response on the console rather than browser).
